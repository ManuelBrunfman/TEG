# Avisos de terceros

## TEG — Tenes Empanadas Graciela

El mapa y las siluetas territoriales utilizadas como base cartográfica proceden del proyecto libre **TEG — Tenes Empanadas Graciela**, cuyo código y recursos se distribuyen bajo GNU General Public License versión 2.

- Proyecto: https://github.com/wfx/teg
- Autor original indicado por el proyecto: Ricardo Quesada y colaboradores.
- Licencia: [LICENSE](LICENSE)

Los recursos se adaptaron para permitir coloreado dinámico por jugador, interacción táctil y resaltado de territorios limítrofes.
